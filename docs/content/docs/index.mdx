NetOpsBench is a closed-loop benchmark for data-center network diagnosis agents: it runs a live SONiC/Containerlab fabric, injects controlled faults, gives agents telemetry-backed evidence, and scores whether they localize the real root cause.

## What NetOpsBench measures

| Benchmark stage | Reader-facing output |
|---|---|
| Generate scenarios and run a topology | Reproducible fault cases with explicit ground truth. |
| Collect Pingmesh and telemetry evidence | A structured `DiagnosticContext` for agents. |
| Invoke a custom diagnosis agent | A `DiagnosisResult` with verdict, fault type, location, evidence, and metadata. |
| Score the diagnosis | Comparable detection, localization, timing, tool-call, and token metrics. |
